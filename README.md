# Plataforma de Tutorías en Python

Proyecto desarrollado para la hackathon de tutorías.

## Descripción
Sistema que permite registrar tutores, registrar estudiantes y agendar tutorías mediante una interfaz gráfica.

## Funcionalidades
- Registrar tutores
- Registrar estudiantes
- Mostrar tutores disponibles
- Agendar tutorías

## Tecnologías usadas
- Python
- Tkinter
- GitHub

## Cómo ejecutar el proyecto

1. Descargar el repositorio
2. Abrir la carpeta del proyecto
3. Ejecutar el archivo principal

python principal.py
